# 04. 테이블 (Table)

## 기본 구조
```html
<table>
  <caption>테이블 제목</caption>
  <thead>
    <tr>
      <th scope="col">이름</th>
      <th scope="col">나이</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>철수</td>
      <td>25</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="2">총 1명</td>
    </tr>
  </tfoot>
</table>
```

## 태그 역할
| 태그 | 역할 |
|------|------|
| `<table>` | 테이블 전체 |
| `<caption>` | 테이블 제목 (접근성, SEO에 중요) |
| `<thead>` | 헤더 영역 |
| `<tbody>` | 데이터 영역 |
| `<tfoot>` | 푸터 영역 (합계, 요약 등) |
| `<tr>` | 행 (table row) |
| `<th>` | 헤더 셀 (굵게 + 가운데 정렬, 의미 있음) |
| `<td>` | 데이터 셀 (table data) |
| `<colgroup>` / `<col>` | 열 그룹화 (열 전체 스타일 적용) |

## 셀 병합
```html
<!-- 가로 병합 (열 합치기) -->
<td colspan="2">2칸 합치기</td>

<!-- 세로 병합 (행 합치기) -->
<td rowspan="2">2행 합치기</td>
```

## colgroup으로 열 스타일
```html
<table>
  <colgroup>
    <col style="width: 200px">  <!-- 1열 -->
    <col style="width: 100px">  <!-- 2열 -->
  </colgroup>
  ...
</table>
```

## scope 속성 (접근성)
```html
<!-- 열 헤더 -->
<th scope="col">이름</th>

<!-- 행 헤더 -->
<th scope="row">철수</th>

<!-- 여러 열/행 헤더 -->
<th scope="colgroup" colspan="2">점수</th>
```
- 스크린리더가 어느 헤더에 속한 데이터인지 파악
- UI 프레임워크에서 테이블 컴포넌트 만들 때 필수

## UI 프레임워크 관점
- `thead`, `tbody`, `tfoot` 구분 → 스크롤 테이블(tbody만 스크롤) 구현 시 필수
- 정렬, 필터, 페이지네이션 기능 붙일 때 `thead`의 `th`가 기준점이 됨
- 대용량 데이터 → 가상 스크롤 고려 필요
- 레이아웃 용도로 table 쓰는 건 안티패턴 → CSS grid/flex로 해야 함

## 주의사항
- `thead`, `tbody`, `tfoot` 없이 `tr`만 써도 동작하지만 명시하는 게 좋음
- `caption`은 테이블 바로 다음에 위치해야 함
