# 09. 링크와 네비게이션

## a 태그 기본
```html
<a href="https://example.com">외부 링크</a>
<a href="/about">내부 링크 (절대경로)</a>
<a href="../about.html">내부 링크 (상대경로)</a>
<a href="#section1">페이지 내 이동 (앵커)</a>
<a href="mailto:test@test.com">이메일</a>
<a href="tel:010-1234-5678">전화</a>
<a href="/file.pdf" download>파일 다운로드</a>
```

## target 속성
| 값 | 역할 |
|------|------|
| `_self` | 현재 탭 (기본값) |
| `_blank` | 새 탭에서 열기 |
| `_parent` | 부모 프레임 |
| `_top` | 최상위 프레임 |

## rel 속성 (보안)
```html
<!-- 새 탭으로 열 때 보안을 위해 noopener noreferrer 권장 -->
<a href="https://example.com" target="_blank" rel="noopener noreferrer">
  외부 링크
</a>
```
- `noopener` → 새 탭에서 `window.opener`로 원래 탭 접근 차단 (탭나이핑 방지)
- `noreferrer` → 참조 페이지 정보(Referer 헤더) 전송 차단
- `nofollow` → 검색엔진에 이 링크를 신뢰하지 말라고 알림 (광고, 외부 링크)

## 앵커 링크
```html
<!-- 이동할 목적지에 id 설정 -->
<h2 id="section1">섹션 1</h2>

<!-- 해당 id로 이동 -->
<a href="#section1">섹션 1로 이동</a>

<!-- 페이지 맨 위로 -->
<a href="#">맨 위로</a>
```

## nav와 함께 쓰기
```html
<nav aria-label="주요 메뉴">
  <ul>
    <li><a href="/">홈</a></li>
    <li><a href="/about">소개</a></li>
    <li><a href="/contact">연락</a></li>
  </ul>
</nav>

<!-- 현재 페이지 표시 -->
<nav>
  <a href="/" aria-current="page">홈</a>
  <a href="/about">소개</a>
</nav>
```
- `aria-current="page"` → 현재 페이지 링크임을 스크린리더에 알림
- `aria-label` → nav가 여러 개일 때 구분

## 절대경로 vs 상대경로
| | 절대경로 | 상대경로 |
|---|---|---|
| 예시 | `/about`, `https://...` | `../about.html`, `./image.jpg` |
| 기준 | 루트(/) 또는 도메인 | 현재 파일 위치 |
| 이동 시 | 영향 없음 | 파일 이동 시 경로 깨질 수 있음 |
