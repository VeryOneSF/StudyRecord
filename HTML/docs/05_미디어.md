# 05. 미디어 (Media)

## 이미지
```html
<img src="image.jpg" alt="이미지 설명" width="300" height="200">
```
- `alt` → 이미지 로드 실패 시 대체 텍스트, 스크린리더, SEO에 필수
- `width`, `height` → 명시하면 브라우저가 공간을 미리 확보 → 레이아웃 밀림(CLS) 방지
- 장식용 이미지는 `alt=""` 빈 문자열로 설정

## 이미지 포맷
| 포맷 | 특징 | 용도 |
|------|------|------|
| JPG | 손실 압축, 용량 작음 | 사진 |
| PNG | 무손실, 투명 지원 | 로고, 아이콘 |
| GIF | 애니메이션 지원, 256색 제한 | 단순 애니메이션 |
| WebP | JPG/PNG보다 용량 작음 | 최신 브라우저 최적화 |
| SVG | 벡터, 확대해도 깨지지 않음 | 아이콘, 로고 |

## 반응형 이미지 (srcset)
```html
<!-- 화면 크기에 따라 다른 이미지 로드 -->
<img
  src="small.jpg"
  srcset="small.jpg 480w, medium.jpg 768w, large.jpg 1200w"
  sizes="(max-width: 480px) 480px, (max-width: 768px) 768px, 1200px"
  alt="반응형 이미지"
>
```
- `srcset` → 브라우저가 화면 크기에 맞는 이미지 선택
- `sizes` → 각 조건에서 이미지가 차지할 크기 힌트

## picture 태그 (아트 디렉션)
```html
<!-- 화면 크기에 따라 완전히 다른 이미지 -->
<picture>
  <source media="(max-width: 480px)" srcset="mobile.jpg">
  <source media="(max-width: 768px)" srcset="tablet.jpg">
  <img src="desktop.jpg" alt="설명">  <!-- 기본값 (필수) -->
</picture>
```

## lazy loading
```html
<!-- 뷰포트에 들어올 때 로드 (성능 최적화) -->
<img src="image.jpg" alt="설명" loading="lazy">

<!-- 즉시 로드 (above-the-fold 이미지에 사용) -->
<img src="hero.jpg" alt="설명" loading="eager">
```

## 비디오
```html
<video controls width="600" poster="thumbnail.jpg">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm"> <!-- 대체 포맷 -->
  브라우저가 video를 지원하지 않습니다.
</video>
```
| 속성 | 역할 |
|------|------|
| `controls` | 재생/정지 컨트롤 표시 |
| `autoplay` | 자동 재생 (muted와 함께 써야 대부분 동작) |
| `muted` | 음소거 |
| `loop` | 반복 재생 |
| `poster` | 재생 전 표시할 썸네일 이미지 |
| `preload` | auto/metadata/none |

## 오디오
```html
<audio controls>
  <source src="audio.mp3" type="audio/mp3">
  <source src="audio.ogg" type="audio/ogg">
</audio>
```

## figure / figcaption
```html
<!-- 이미지에 설명을 붙일 때 -->
<figure>
  <img src="chart.png" alt="2024년 매출 차트">
  <figcaption>2024년 매출 현황</figcaption>
</figure>
```
- `figure` → 이미지, 코드, 표 등 독립적인 콘텐츠 묶음
- `figcaption` → figure의 설명, alt와 중복되지 않게 작성
